Python API
==========

The section demonstrates how to install and use the Python API.
Jupyter notebooks are used to analyze data and save results back the server.

Follow the instructions in the
:download:`README.md <https://github.com/ome/omero-guide-python/blob/master/README.md>` file to run the notebooks either locally or on `mybinder.org <https://mybinder.org/v2/gh/ome/omero-guide-python/master?filepath=notebooks>`_.
